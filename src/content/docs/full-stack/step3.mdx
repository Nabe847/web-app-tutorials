## このステップについて

このステップでは、アプリケーションの動作確認を行います。

## 動作確認

### 1. サーバーの起動

1. バックエンドサーバーを起動：
```bash
cd task-manager-backend
uvicorn app.main:app --reload
```

2. フロントエンドサーバーを起動：
```bash
cd task-manager-frontend
npm run dev
```

### 2. 機能テスト

ブラウザで `http://localhost:3000` を開き、以下の機能をテストします：

1. **Todoの作成**
   - 新しいTodoを入力
   - 追加ボタンをクリック
   - リストに表示されることを確認

2. **Todoの完了状態の切り替え**
   - チェックボックスをクリック
   - 完了状態が切り替わることを確認

3. **Todoの削除**
   - 削除ボタンをクリック
   - リストから消えることを確認

### 3. エラーハンドリングの確認

1. **バックエンドサーバーを停止**
   - ターミナルで `Ctrl+C` を押してサーバーを停止
   - フロントエンドで操作を試みる
   - エラーメッセージが表示されることを確認

2. **バックエンドサーバーを再起動**
   - サーバーを再起動
   - フロントエンドの操作が正常に動作することを確認

## よくある問題と解決方法

**Todoが表示されない場合**

- 解決方法：
  1. バックエンドサーバーが起動しているか確認
  2. ブラウザの開発者ツールでエラーを確認
  3. CORSの設定が正しいか確認

**Todoの操作が失敗する場合**

- 解決方法：
  1. バックエンドサーバーが起動しているか確認
  2. ブラウザの開発者ツールでエラーを確認
  3. APIリクエストの内容を確認
