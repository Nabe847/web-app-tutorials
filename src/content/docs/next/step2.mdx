## このステップで学ぶこと

前回の環境構築に続いて、今回は実際にページを作成していきます。シンプルな内容から始めて、少しずつ機能を追加していきましょう。

- トップページの編集
- 基本的な JSX の書き方
- インラインスタイルの適用方法
- ブラウザの自動更新機能の確認

### 用語の説明

- **JSX**: JavaScript の中で HTML のような記法を使って UI を記述する方法です。例えば `<div>` や `<h1>` などの HTML タグを JavaScript の中で直接使うことができます。
- **インラインスタイル**: HTML 要素に直接スタイルを適用する方法です。このステップでは `style={{ }}` という書き方で、要素の見た目（余白や幅など）を設定します。

### app/page.js を編集

シンプルなトップページを作成します：

```javascript
// 1. コンポーネントの定義
export default function Home() {
  return (
    // 2. 画面の見た目を定義（HTMLのような記法で記述）
    <div 
      // 3. インラインスタイルの適用
      style={{ padding: '20px', maxWidth: '600px', margin: '0 auto' }}
    >
      <h1>タスク管理アプリ</h1>
      <p>ここにタスクを表示します</p>
    </div>
  );
}
```

### コードの解説

1. コンポーネントの定義
   ```javascript
   export default function Home() {
     return (
       <div style={{ padding: '20px', maxWidth: '600px', margin: '0 auto' }}>
         <h1>タスク管理アプリ</h1>
         <p>ここにタスクを表示します</p>
       </div>
     );
   }
   ```
   - `export default` は、この関数を他のファイルから使えるようにするための記述です
   - `Home` はコンポーネント名で、必ず大文字で始める必要があります

2. 画面の見た目を定義
   ```javascript
   <div style={{ padding: '20px', maxWidth: '600px', margin: '0 auto' }}>
     <h1>タスク管理アプリ</h1>
     <p>ここにタスクを表示します</p>
   </div>
   ```
   - JSX を使って UI を記述しています
   - `<div>` や `<h1>` などの HTML タグを使って、ページの構造を作ります

3. インラインスタイルの適用
   ```javascript
   style={{
     padding: '20px',
     maxWidth: '600px',
     margin: '0 auto'
   }}
   ```
   - `padding: '20px'`: 内側の余白を 20px に設定
   - `maxWidth: '600px'`: 最大幅を 600px に制限
   - `margin: '0 auto'`: 左右の余白を自動で調整して中央寄せ

### 変更の確認方法

1. ファイルを保存すると、ブラウザが自動的に更新されます
2. `http://localhost:3000` にアクセスして、変更が反映されているか確認してください

### よくある問題と解決方法

**変更が反映されない場合**

- ファイルが正しく保存されているか確認
- 開発サーバーが正常に動作しているか確認
- ブラウザを手動で更新（F5 キー）してみる

**エラーが表示される場合**

- コードの書き方が正しいか確認
- 中括弧 `{}` や括弧 `()` の閉じ忘れがないか確認
- セミコロン `;` の付け忘れがないか確認

### 💡 ヒント

- スタイルは `style={{ }}` の中に JavaScript オブジェクトとして記述します
- コンポーネント名は必ず大文字で始める必要があります
- ファイルを保存すると、ブラウザが自動的に更新されます（ホットリロード機能）

### 🎯 理解度チェッククイズ

以下の質問に答えて、学習内容を確認しましょう！

1. **JSX でスタイルを適用する正しい書き方は？**

   - A: `style="padding: 20px"`
   - B: `style={{ padding: '20px' }}`
   - C: `style={padding: 20px}`
   - D: `style={padding: '20px'}`

   <details>
   <summary>答えを見る</summary>

   **答え: B: `style={{ padding: '20px' }}`**
   - JSX では、スタイルを JavaScript オブジェクトとして記述します
   - 外側の `{}` は JSX の式を表し、内側の `{}` は JavaScript オブジェクトを表します
   </details>

2. **ページを中央寄せにするスタイルは？**

   - A: `margin: center`
   - B: `margin: 0 auto`
   - C: `align: center`
   - D: `text-align: center`

   <details>
   <summary>答えを見る</summary>

   **答え: B: `margin: 0 auto`**
   - `margin: 0 auto` は、上下の余白を 0 に、左右の余白を自動で調整して中央寄せにします
   - これは要素を水平方向に中央に配置する一般的な方法です
   </details>

3. **JSX で HTML タグを書く際の注意点は？**

   - A: タグは必ず小文字から始まる（例：`<div>`）
   - B: タグは必ず大文字から始まる（例：`<Div>`）
   - C: タグは HTML と同じように書く（例：`<div>`）
   - D: タグは JavaScript の構文で書く

   <details>
   <summary>答えを見る</summary>

   **答え: C: タグは HTML と同じように書く（例：`<div>`）**
   - JSX では、通常の HTML タグ（`<div>`、`<h1>` など）は HTML と同じように書きます
   - コンポーネント名（このステップでは使用しません）の場合のみ大文字から始まります
   </details>
