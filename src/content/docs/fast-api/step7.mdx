## Delete機能の実装

`app/main.py` に以下のコードを追加して、Todoの削除機能を実装します：

```python
# Todoの削除
@app.delete("/todos/{todo_id}")
def delete_todo(todo_id: int, db: Session = Depends(get_db)):
    db_todo = db.query(models.Todo).filter(models.Todo.id == todo_id).first()
    if db_todo is None:
        raise HTTPException(status_code=404, detail="Todo not found")
    db.delete(db_todo)
    db.commit()
    return {"message": "Todo deleted successfully"}
```

> 💡 **DELETE**: リソースの削除に使用されるHTTPメソッドです。

## APIの動作確認

uvicorn app.main:app --reload

2. ブラウザで `http://127.0.0.1:8000/docs` を開きます。

3. Todoの削除（DELETE `/todos/{todo_id}`）を試してみましょう。

## よくある問題と解決方法

**404エラーが発生する場合**

- エラーメッセージ例：`{"detail": "Todo not found"}`
- 解決方法：
  1. 指定したIDのTodoが存在するか確認
  2. 正しいIDを指定しているか確認
